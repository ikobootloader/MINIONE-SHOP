# Minione Shop - Vitrine Bohème Chic

Page vitrine e-commerce convertie de React/JSX vers HTML/CSS/JS vanilla.

## Structure

- `index.html` - Structure de la page
- `styles.css` - Styles complets (responsive)
- `script.js` - Menu mobile toggle
- `minione_vitrine_broadcast_style.jsx` - Source React d'origine

## Caractéristiques

- Responsive (mobile-first)
- Navigation sticky
- Animations CSS
- Aucune dépendance externe
- Images via Unsplash

## Sections

1. Bannière promotionnelle
2. Header avec navigation
3. Hero (visuel + texte)
4. Catégories (3 cards)
5. Nouveautés produits (4 items)
6. Lookbook
7. Features (4 icônes)
8. Footer

## Utilisation

Ouvrir `index.html` dans un navigateur.
